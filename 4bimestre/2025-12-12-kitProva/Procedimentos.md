Seguir esses passos para criar crud

# Lembre-se de manter tudo padronizado 

1) Crie uma pasta para conter seu projeto
2) Abra essa pasta no vscode
3) Deverão estar na pasta, no mínimo 3 arquivos
   1) classe (nomeCrud.js)
   2) html (nomeCrud.html)
   3) controle (nomeCrudControle.js)
4) Copie de outro projeto a classe, ajuste o nome do arquivo e o conteúdo
5) Copie de outro projeto o arquivo html, ajuste conforme for necessário. Tome cuidado com os tipos (text, number, date, checkbox, etc). Ajuste os imports (para que "ache os arquivos da classe e de controle). Cuide com os nomes dos atributos, padronize para ficar mais fácil. 
6) Copie o arquivo de controle. Renomeie. 
7) Faça os ajustes necessários.
8) Verifique as restrições e teste.
9) Faça as funcionalidades e teste.
